# Goofy-screener-phase3
Downloads live price data for ~120 stocks across all 3 markets Runs a full parameter grid search on each strategy (train: 2016–2020) Validates the best strategy out-of-sample (test: 2021–present) Scores each asset 0–100 using Sharpe, Return, Max Drawdown, and DD Saved vs B&amp;H Ranks into tiers: S/A/ B/Skip. Saves a colour-coded multi-tab Excel report
